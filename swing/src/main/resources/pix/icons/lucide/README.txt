Steps:
1. Search for the right icon on `https://lucide.dev/icons`
2. Set the following settings on the site:

	Color: 			#000000
	Stroke width:		1.25px
	Size: 			16px
	Absolute stroke width:	Disabled

3. Download the SVG
4. Verify that the SVG is correct. You may have to open the SVG file in a text editor and modify the stroke color to be `stroke="#000000"`;
sometimes, it doesn't export well.

If you need to modify the SVG, we recommend doing this in Inkscape.
